# Tuluminati VAPI

**Voice AI concierge for Tulum, Mexico. VAPI-powered voice agent with Luna avatar providing local recommendations, booking assistance, and area guides.**

## Features

- VAPI voice agent with natural conversation
- Luna avatar with visual presence
- Tulum area guides and local recommendations
- Cloudflare Worker backend
- OpenClaw swarm integration

## Architecture

```
index.html            Landing page with Luna avatar
vapi/                 VAPI voice agent configuration
worker/               Cloudflare Worker for backend logic
openclaw/             Swarm coordination layer
images/               Visual assets
```
